MISSION STATEMENT

MADE IN 'olu'olu（オルオル）

私たち'olu'olu（オルオル）は、2000年の創業以来、ウェブとインタラクティブの分野を中心に、多岐にわたる領域、表現、技術で、企業や社会の「aid＝助力」になるべく、数多くの仕事を手がけてきました。

かつては革新的なウェブサイトをつくれば解決することができた多くの課題。しかし、複雑な変化と進化を繰り返す世界の中で、今や、単独のアイデアや技術だけで完遂できるプロジェクトはほとんどありません。

空間やジャンル、デバイスの境界を越えた視点でものをつくる。その中から真の価値を見つけ出し、新しい体験や関係性、具体的な機能や行動を生み出す。

こうしたプロセスを自ら開発して、正しいクリエイティブのかたちを追求し続けることこそが、私たちの使命です。 私たちはこれからも、クライアントと向き合い、ブランドと向き合い、課題と向き合い、創造と向き合い、ビジネスと向き合い、何よりも「つくること」とまっすぐに向き合って、この世界を変えていきます。

    PROFILE

    1995年生まれ。

    'olu'olu 代表
    Web Designer / Web Writer

    大学在学中にアメリカへ留学。その際に、3ヶ月宿無しでアメリカ全土を旅をし、旅の魅力に圧倒され卒業後も旅しながら仕事をすると決意。卒業後に未経験からフリーランスライターになり、セブ島で旅しながら仕事をし夢を実現。

    セブ島でライターをしながら旅をした結果、部族の集落にたどり着き、そこで部族とともに生活をしながらライターの仕事をすることに。ここでの経験がこれまでの価値観を大きく変える結果に。

    ライターになる前にプログラミング学習をしていたことから、Webデザインのキャリアに進みたいと思い始め、再び学習を再開し、本格的にこの道へ。
