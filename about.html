    PROFILE

    1995年生まれ。

    'olu'olu -オルオル-

    Web Writer
    Web / Graphic Designer

    大学在学中にアメリカへ留学。その際に、3ヶ月宿無しでアメリカ全土を旅をし、旅の魅力に圧倒され卒業後も旅しながら仕事をすると決意。卒業後に未経験からフリーランスライターになり、セブ島で旅しながら仕事をし夢を実現。

    セブ島でライターをしながら旅をした結果、部族の集落にたどり着き、そこで部族とともに生活をしながらライターの仕事をすることに。ここでの経験がこれまでの価値観を大きく変える結果に。

    ライターになる前にプログラミング学習をしていたことから、Webデザインのキャリアに進みたいと思い始め、再び学習を再開し、本格的にこの道へ。
